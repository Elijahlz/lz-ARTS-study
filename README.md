# 耗子叔ARTS挑战

## 说明

ARTS 是 Algorithm、Review、Tip、Share 的简写

1.算法：每周至少做一个本文给出的算法题  
2.评论：阅读并点评至少一篇英文技术文章  
3.提示：学习至少一个技术技巧  
4.分享：分享一篇有观点和思考的技术文章
